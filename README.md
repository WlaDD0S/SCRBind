# SCRBind

## Общая информация
- Автоматизирует рутину.
- Облегчает комуникацию.

- Работает только на Full HD экране (возможно потом адаптирую под другие)
- Требуется **AutoHotkey v2.0** (для не скомпилированной версии).

## Роли и сообщения

| Роль | Сообщения | Автоматизировано |
|------|-----------|-----------------------|
| GD   | "Someone need GD?", "Nearest station?", "[DS ST] Safe shift!" | Прибытие и отправка с диспетчером и без него |
| DS   | "[HC] Safe trip!", "[HC] Safe trip & GD!" | Нажатие "TRTS", "CD", "RA" |
| QD   | "Free GD?", "Nearest station - ST", "[DS ST] Safe shift!" | Всё связанное с дверями и охранником, а также AWS и SPAD |
| SG   | "Zone # on manual control!", "Zone # on automatic control!", "[DS ST] please dispatch HC!" | Ничего |


## Горячие клавиши
- **F7** — отправляет выбранное сообщение.
- **F6** — скрывает/показывает GUI.
- **F3** — быстро выключает.
- **F1** — включает/отключает возможность перемещения GUI.

**Переназначение клавиш** — Открываете .ahk файл в любом текстовом редакторе и изменяете нужные кнопки (EXE файл нельзя перенастроить).


## Функции
1. Выбор роли запускает или останавливает автоматизацию.
2. Выбор сообщения или станции изменяет сообщение которое потом отправится.
3. Сообщения отправляются по нажатию F7.
4. GUI можно скрыть/показать (F6) и перемещать (F1).
5. Быстрое отключение/включение скрипта — F3.
